# Mintsys

Infomaniak Domains:

- mintsys.ch

Mintsys die kurze From von [[Mint System]].

## Alternative Domain

Die Domain `mintsys.ch` wird als alternative Domain für Dienste von Kunden verwendet. Insbesondere dann, wenn Kunde keine eigene Domain haben oder nicht zur Verfügung stellen wollen. Beispiele:

* kaminfeger.mintsys.ch
* tellspiele.mintsys.ch

Mit [[Mailgun]] können auch Absender-Adressen wie `tellspiele@mail.mintsys.ch` eingerichet werden.
