## Entity-Relationship-Model erstellen

Arbeitsschritte:

- In Nextcloud ein [[draw.io]] ein ERM-Diagramm erstellen
- Entities und Attribute modellieren
- Beziehungen zwischen Entities definieren

## User Stories erstellen

Arbeitsschritte:

- Erstellen Dokument anhand [[Vorlage User Stories]]
- Ein erstes Beispiel erstellen
- Dokument mit Kunde teilen

## User Stories mappen

Arbeitsschritte:

- Anhand der Stories eine Modulstruktur definieren
- Erstellen Tabellen mit User Story und dazugehörigem Modul

## Spezifikation erstellen

Arbeitsschritte:

- Erstellen Dokument anhand [[Vorlage Spezifikation Applikations-Modul]] in Ablage von Kunde
- Erarbeiten Spezifikation anhand User Stories
- Validierung der Spezifikation mit Kunde
- Erstellen Aufgabe mit Link auf die Spezifikation
- Freigabe für Entwicklung erteilen
