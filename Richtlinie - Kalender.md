# Richtlinie - Kalender

## Standardmässig Offen

Alle Mitarbeitenden haben Zugriff auf die Kalender von anderen.

## Private Termine

Private Termine werden in einem separaten Kalender geführt oder als im Arbeitskalender als privat markiert.
