Wir kombinieren [[Jupyter]] und [[OpenCode]] um einen Workspace zur Arbeit mit Agenten bereitzustellen.

## Vergleich mit [[Odoo KI]]

|                                | **Odoo AI**              | **Jupyter Lab x OpenCode**                                   |
| ------------------------------ | ------------------------ | ------------------------------------------------------------ |
| Zugriff auf Odoo               | Direkt (Odoo-ORM)        | API (XML-RPC und JSON-API)                                   |
| Art des Zugriffs               | Lesen                    | Lesen und Schreiben                                          |
| Verfügbare Models              | Gemini und Claude        | [OpenCode Providers](https://opencode.ai/docs/de/providers/) |
| Dialog                         | Mit Odoo Chat            | Mit Jupyter Lab Chat                                         |
| Technische Anforderung an Odoo | Postgres mit PGVector    | -                                                            |
| Anwendungsfall                 | Navigation und Reporting | Datenverarbeitung und Automatisierung                        |
| Demo                           | http://demo.odoo.com/    |                                                              |
| Inference Kosten               | Odoo S.A. bezahlt        |                                                              |
