## Archivierung durchführen

Arbeitsschritte:

- Nextcloud Root-Folder in Terminal öffnen

```bash
cd ~/cloud.mint-system.ch/Mint\ System
```

- Den Archivierungs-Befehl ausführen

```bash
task move-archived-files
```
