Server bei [[Kreis Infrastruktur]] bestellen.

## Managed Server bestellen

Arbeitsschritte:

- Zusammenstellen Informationen Kunden:
	- **Alias**: Name des Servers. Beispiel: `server.mint-system.com`
	- **Server-Typ**: Server-Typ aus dem Angebot.
	- **Betriebssystem**: Das [[Betriebssystem#Server]] bestimmen. Empfohlen ist [[Debian]].
	- **DNS-Verwaltung**: Wer muss zum Einrichten der DNS-Einträge kontaktiert werden
- Erstellen Aufgabe "Server bereitstellen" in Kundenprojekt mit Kundeninformationen
- Aufgabe an [[Kreis Infrastruktur]] zuweisen
