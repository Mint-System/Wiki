## Produktives Upgrade planen

Arbeitsschritte:

- Mit dem Kunden einen Termin für das produktive Upgrade vereinbaren.
- Festhalten wer von Seite Kunde kontaktiert werden soll.
- Termin in Kalender erstellen.

## Produktives Upgrade durchführen

Arbeitsschritte:

- Nachricht an den Kunden schicken:

```
Guten Tag

Diese Odoo Installation wird innerhalb der nächsten Stunde von Version 15.0 auf 17.0 aktualisiert.

Wir bitten euch während dieser Zeit keine Bearbeitungen in Odoo vorzunehmen. Alle Änderungen ab jetzt werden nicht in der aktualisierten Odoo Installation sein.

Mit freundlichen Grüssen

Mint System
```

- Produktions-Upgrade gemäss dem Playbook ausführen.
- Nachricht an den Kunden bei erfolgreichem Upgrade:

```
Guten Tag

Das Odoo Upgrade war erfolgreich.

Den Odoo Upgrade Bericht findet man unter "Dialog > Kanäle > Administrators".

Wir bitten euch die Odoo Installation zu verfizieren und allfällige Fehler zu melden.

Mit freundlichen Grüssen

Mint System
```
